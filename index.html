Минимальные требования¶

OC: Windows 7 и выше

Процессор: не менее Intel Pentium G4650

RAM: 2GB

Библиотеки и требуемые программы: .NET Framework 4.7, Microsoft Word

Установка¶

1. Скачать программу по ссылке: ссылка

2. Распаковать архив

3. Запустить файл "WindowsFormsApp4.exe", который лежит в корне каталога

4. Сделано

Использование¶

Авторизация

1. Записать в текстовое поле под стрелкой 1, ваш логин

2. Записать в текстовое поле под стрелкой 2, ваш пароль

3. Ввести соотвествующие данные

4. Нажать кнопку "Войти"

5. Готово, вы вошли под свой учетной записью

Создание тестов

1. Сначала записать в текстовом поле под стрелкой 1 название теста

2. Нажать на кнопку "Добавить новый тест""

3. Готово, вы создан тест

Добавление вопрос для тестов

1. Вам откроется окно с добавлением вопросов

2. Запишите в текстовое поле под стрелкой один, сам вопрос

3. В оставшиеся поля введите ответы

4. Под стрелкой 2 выберите правильный ответ

5. Нажмите на кнопку "Добавить вопрос"

6. Когда закончите с добавлением вопросов нажмите "Сохранить тест"

Удаление тестов

1. Выбрать тест из таблицы

2. Нажать на кнопку "Удалить"

3. Нажать кнопку "ОК"

4. Готово, вы удалили тест

Редактирование тестов

1. Нажать на кнопку "Редактирование тестов"

2. Выбрать из таблицы нужный тест

3. Нажать на кнопку "Редактирование"

4. Перейдете на окно Редактирование

5. Выбрать вопрос из таблицы

6. В текстовых полях появится данные вопроса

7. Когда отредакирвоали вопрос,нажать на "Редактировать вопрос"

8. Потом нажать на "Сохранить тест"
